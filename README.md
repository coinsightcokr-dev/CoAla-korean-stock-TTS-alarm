# CoAla-korean-stock-TTS-alarm

한국 주식 가격 추적 TTS 음성 알림 프로그램 겸 모바일 알람 서버

한국 주식에 대해 한 번의 알람 설정으로

실시간 가격 추적 및 연속적인 알람을

TTS 음성으로 제공합니다.

차트를 계속 보고 있지 않아도 

음성으로 제공되는 연속 알람을 통해

주가의 변동 흐름을 따라갈 수 있습니다.

-------------------------------------
특징

1. 한번의 알람 설정으로 가격을 추적하며 연속적으로 알람을 제공합니다.

- 가격 추적: 변동 차이 값(gap)을 설정하면 차이가 발생함에 따라 연속적으로 알람을 제공합니다.

예: 70000원 주가에 100원의 차이값을 설정하는 경우

70000 > 100 상승 >70100 상승 알람 발생 > 100 상승 > 70200 상승 알람 발생 > 100 하락> 70100 하락 알람 발생 > 100 상승 > 70200 상승 알람 발생 >......

- 기준값 대비  가격 차이를 알려줍니다.

  예: 100000원, 기준값 대비 +1000원


2. TTS 기능을 사용하여 음성으로 제공됩니다.

 - 알람 내용을 알람별로 개별화 가능합니다. 
 
 예: 삼성전자 100000원 상승. 100000원 업, 

3. 모바일 연동 가능 (전용 앱 필요)


-------------------------------------
프로그램 설명

1. 최초 실행 시 약관 동의 필요.

2. 종목코드 - 한국 주식의 종목 코드를 입력하고 조회를 클릭합니다. 예: 005930 (삼성전자 종목 코드)

3. 최근조회 - 최근에 조회한 종목 목록이 10개까지 표시됩니다. 여기서 종목을 클릭하면 바로 다시 조회 가능합니다.

4. gap - 알림 간격을 설정합니다.

   현재가가 70000원이고 gap이 100 인 경우 하단 69900, 상단 70100 에 알람이 설정됩니다.

   이후 가격 도달 시 gap 만큼 변경되어 새로 알람이 자동으로 설정됩니다.

  예: 100 상승 >70100 상승 알람 발생 > 하단 70000 알람 자동 생성, 상단 70200알람 자동 생성
      
      다시 100 상승 > 70200 상승 알람 발생 > 하단 70100 알람 자동 생성, 상단 70300알람 자동 생성
      
      100 하락> 70100 하락 알람 발생 > 하단 70000 알람 생성, 상단 70200알람 생성......

5. 기준가 - 기준이 되는 가격을 설정합니다. 예를 들면 매수가를 입력할 경우 현재 가격이 기준가 대비 얼마 변동이 있는지 알려줍니다.

  예: 기준가 60000원 입력시 "상승 70000원 기준가 대비 +10000원" 으로 현재 손익을 알려줍니다.

6. TTS 알람문구 - 변수를 사용하여 TTS 알림 내용을 수정할 수 있습니다.

  예: 한 종목만 알림을 할 경우 종목명({name}을 삭제하면 가격만 알려줍니다.
  
  예: "기준가 대비" 를 "차이"로 변경하거나 다른 용어로 변경하면 그대로 읽어줍니다.
   
   -tts 테스트 버튼을 누르면 어떻게 발음되는지 들어볼 수 있습니다.

7. 알림 방식 - TTS 실행 플랫폼 선택

   예: 윈도우 tts에 체크된 경우 컴퓨터에서 TTS가 작동합니다.

   예: CoAla 앱에 체크된 경우 모바일 앱 Coala에서 TTS가 작동합니다.

   얘: 둘 다 체크가 안 된 경우 TTS 작동 없이 알람 로그에만 표시됩니다.

7.1 CoAla 앱 체크된 경우 Device ID와 동기화 코드가 필요합니다.
   
   Device ID는 CoAla 앱의 하단 메뉴에서 내 정보를 클릭하면 상단에 표시됩니다.
   
   공유 아이콘을 클릭하여 메일이나 메신저로 보낸 뒤 컴퓨터에서 붙여넣으십시오.
   
   동기화 코드는 숫자 4자리 이상으로 앱과 프로그램에서 같은 숫자를 넣으시면 됩니다.
   
   동기화 코드 기반으로 모든 내용은 암호화되어 서버로 전송 및 모바일로 전달됩니다.
   
   서버에는 동기화 코드가 전달, 저장되지 않아 서버 차원에서 내용을 확인할 수 없습니다.
   
   프로그램을 종료하면 앱으로 알람을 보내지 않습니다.

7.2 앱 사용 보충
   
   https://play.google.com/store/apps/details?id=kr.co.coinsight.coala
   
   코인, 주식 등 모든 유형에서 1개의 알람은 아무 제한 없이 무료입니다.
   
   1개 이상의 알람은 프리미엄 또는 구독이 필요합니다.
   
   광고 1개 시청시 2시간의 프리미엄이 제공됩니다. 점심시간 등 일시적 사용에는 광고 1번 시청으로 충분합니다.
   
   1일 20개까지 광고 시청이 가능합니다.

8. 알람 추가를 누르면 하단에 알람 카드가 등록됩니다. 일시정지, 재시작, 삭제 등 작업을 할 수 있습니다.

   앱을 사용하는 경우 앱에서의 일시정지, 재시작, 삭제는 별도로 작동합니다.

   예: 프로그램에서 실행하더라도 앱에서 일시정지를 하면 앱에서 알람이 작동하지 않습니다.

   예: 프로그램에서 정지(또는 프로그램 종료)시 앱은 실행은 되지만 알람이 전달되지 않습니다.

   예: 프로그램에서 삭제하더라도 앱에서는 알람이 등록되어 있습니다. 별도로 앱에서 삭제해야 합니다. 

9. 알람 로그에 알람에 대한 로그가 기록됩니다.
